#터미널 커맨드
ls          → 기본 목록
ls -l       → 상세 정보 포함 (권한, 크기, 날짜)
ls -a       → 숨김 파일(.으로 시작하는 것)까지 포함
ls -la      → 상세 + 숨김 파일 다 보기 (가장 많이 씀)

실행 명령어

번호	수행해야 할 것	사용 명령어
1	현재 위치 확인	pwd
2	목록 확인 (숨김 파일 포함)	ls -la
3	폴더 이동	cd
4	폴더/파일 생성	mkdir, touch
5	파일 복사	cp
6	파일 이동/이름 변경	mv
7	파일/폴더 삭제	rm
8	파일 내용 확인	cat
9	빈 파일 생성	touch
